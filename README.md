# Bangalore House Price Prediction

A Machine Learning web application that predicts house prices in Bangalore based on user inputs.

## Features

- Predicts Bangalore house prices
- User-friendly web interface
- Trained using Machine Learning
- Built with Flask

## Technologies Used

- Python
- Flask
- Scikit-learn
- NumPy
- Pandas
- HTML
- CSS
- JavaScript

## Project Structure

```
Price_prediction/
│── client/
│── model/
│── server/
│── requirements.txt
│── .gitignore
│── README.md
```

## Installation

1. Clone the repository

```bash
git clone https://github.com/Vanshikaaa49/Price_prediction.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
python server.py
```
